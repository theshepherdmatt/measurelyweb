Simple, No-Internet Setup

The Measurely box creates its own local WiFi. No router passwords. No cloud. No app. No accounts. No internet required.

  1. Power on the Measurely box
  2. Connect to WiFi: MeasurelyConnect
  3. Scan the QR code
    or go to measurely.local:5000 in your browser
  4. Run the sweep and Measurely takes over

Use It On Any Device
